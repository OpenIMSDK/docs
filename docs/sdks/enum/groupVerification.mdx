# GroupVerification

:::info

进群验证选项。

:::

## 进群验证选项 {#GroupVerification}

| 枚举值 | 描述                                 |
| ------ | ------------------------------------ |
| 0      | 申请进群需要群主或管理员同意；群成员邀请可以直接进群              |
| 1      | 申请进群需要群主或管理员同意；群主和管理员邀请可以直接进群；普通成员邀请进群需群主或管理员同意 |
| 2      | 直接进群                             |
