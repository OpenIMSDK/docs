# RecvMsgOpt

:::info

在[用户信息](docs/sdks/class/user/userInfo.mdx)的`globalRecvMsgOpt`中，全局控制是否接收离线推送。  
在[会话](docs/sdks/class/conversation/conversationInfo.mdx)中的`recvMsgOpt`除了控制是否接收该会话离线推送外，还会控制该会话未读数是否计入总未读数。

:::

## 会话消息接收选项

| 枚举值 | 描述                                                     |
| ------ | -------------------------------------------------------- |
| 0      | 正常接受消息                                             |
| 1      | 保留字段                                                 |
| 2      | 接收消息，但无离线推送。在会话上时，该会话不计入总未读数 |
