# GroupMemberFilter

:::info

筛选成员类型。

:::

## 筛选成员选项 {#GroupMemberFilter}

| 枚举值 | 描述               |
| ------ | ------------------ |
| 0      | 所有成员           |
| 1      | 群主               |
| 2      | 群管理员           |
| 3      | 群普通成员         |
| 4      | 群管理员和普通成员 |
| 5      | 群主和群管理员       |
