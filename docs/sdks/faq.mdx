# 常见问题

## 发送消息是否需要先获取会话？

在发送消息时，无需预先获取会话相关的参数。具体操作如下：

- **单聊**：仅需提供 `userID` 参数。
- **群聊**：仅需提供 `groupID` 参数。

## 何时调用 `getOneConversation`？

用户在客户端发起聊天时，在聊天窗口的UI界面上，一般会展示会话相关信息。

1. **如果用户从会话列表中选择某个会话发起聊天**：
     - 会话信息已包含在会话列表中，无需额外调用 `getOneConversation`。
2. **如果用户从其他入口发起聊天**：
     - 需要先调用 `getOneConversation` 来获取会话信息。
3.  **在聊天 UI 界面上展示会话相关信息。**
    - 根据产品需求，可把会话信息展示在聊天 UI 界面上。
