# createFileMessageFromFullPath

## 功能介绍

:::info 说明

创建文件消息，由SDK内部根据绝对路径提取文件信息并创建文件消息。

:::

<TabItem value="uni-app">

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('createFileMessageFromFullPath', operationID: string, {
  filePath: string,
  fileName: string
}): Promise<MessageItem>
```

### 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述                                                    |
| ----------- | -------- | -------- | ------------------------------------------------------- |
| operationID | string   | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |
| filePath    | number   | 是       | 文件绝对路径                                            |
| fileName    | string   | 是       | 文件名称                                                |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                         | 描述         |
| --------------- | ---------------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<[MessageItem](docs/sdks/class/message/messageInfo.mdx)\> | 调用成功回调 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/class/response.mdx)\>          | 调用失败回调 |

### 代码示例

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

IMSDK.asyncApi('createFileMessageFromFullPath', IMSDK.uuid(), {
  filePath: '',
  fileName: '',
})
  .then((data) => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
