# deleteMessageFromLocalStorage

## 功能介绍

:::info 说明

从本地删除一条消息，卸载 APP 后会重新获取到。

:::

:::caution 注意

**相关回调**：      
[onConversationChanged](docs/sdks/callback/onConversationChanged.mdx)   
如果删除的是最新一条消息，则会话最新一条消息会改变

:::
