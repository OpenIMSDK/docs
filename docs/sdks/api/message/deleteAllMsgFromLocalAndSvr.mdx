# deleteAllMsgFromLocalAndSvr

## 功能介绍

:::info 说明

删除本地和服务器所有会话中的消息，会话仍会保留。

:::

:::caution 注意

**相关回调**：      
[onConversationChanged](docs/sdks/callback/onConversationChanged.mdx)

:::
