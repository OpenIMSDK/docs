# setMessageListener

## 功能介绍

:::info 说明

设置批量触发消息事件监听器，对于新消息进行批量推送，设置后`advancedMsgListener`中的`onRecvNewMessage`和`onRecvOfflineNewMessage`回调不再触发。

:::

:::caution 注意

（1）在 initSDK 之后，login 之前调用；  
（2）只能调用一次。

:::
