# subscribeUsersStatus

## 功能介绍

:::info 说明

订阅指定用户的在线状态，并返回其在线状态。

:::

:::caution 注意

每个用户订阅人数不能超过3000，如超过则按照订阅时间淘汰。


**相关回调**:    
[onUserStatusChanged](../../callback/onUserStatusChanged)  


:::
