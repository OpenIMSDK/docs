# getUsersInfoWithCache

## 功能介绍

:::info 说明

获取指定用户的个人信息，如果指定了群组 ID，还会获取在群组内的群成员信息。

:::

:::caution 注意

（1）建议一次最大 100 个，因为数量过多可能会导致数据包太大被后台拒绝；   
（2）优先从服务端获取，如果失败，则从本地缓存中获取；   
（3）用户调用此接口获取非好友的个人信息，SDK 内部会更新此缓存。

:::
