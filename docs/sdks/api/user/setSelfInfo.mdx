# setSelfInfo

## 功能介绍

:::info 说明

修改当前登录用户的个人信息。包含Ex字段。   

:::

:::caution 注意




**相关回调**:    
[onSelfInfoUpdated](docs/sdks/callback/onSelfUserInfoUpdate.mdx)  


:::

| 参数名称        | 参数类型                                                | 描述         |

IMSDK.asyncApi('setSelfInfo', IMSDK.uuid(), userInfo)
