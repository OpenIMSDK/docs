# addBlack

## 功能介绍

:::info 说明

把对方添加到黑名单中。

:::

:::caution 注意

意味着对方不能给你发送消息，但你可以给对方发送消息。  
**相关回调**:    
[onBlackAdded](../../callback/onBlackAdded) 


:::

IMSDK.asyncApi('addBlack', operationID: string, {

| ----------- | -------- | -------- | ------------------------------------------------------- |

| --------------- | ------------------------------------------------------- | ------------ |
