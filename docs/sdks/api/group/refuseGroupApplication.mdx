# refuseGroupApplication

## 功能介绍

:::info 说明

拒绝进群申请。

:::

:::caution 注意
（1）此群的群主或群管理员才可调用；     
（2）进群请求被拒绝后，管理员和群主无法再操作。

**相关回调**:    
[onGroupApplicationRejected](../../callback/onGroupApplicationRejected)  


:::
