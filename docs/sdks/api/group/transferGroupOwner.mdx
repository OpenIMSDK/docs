# transferGroupOwner

## 功能介绍

:::info 说明

转让群主身份。

:::

:::caution 注意

（1）每个群只能有一个群主；   
（2）调用者必须是此群的群主。     

**相关回调**:    
[onGroupInfoChanged](docs/sdks/callback/onGroupInfoChanged.mdx)   
[onGroupMemberInfoChanged](docs/sdks/callback/onGroupMemberInfoChanged.mdx)


:::
