# setGroupInfo

## 功能介绍

:::info 说明

设置群信息，包括群头像、名称、公告、简介、扩展字段等。

:::

:::caution 注意

管理员和群主才有设置权限。


**相关回调**:    
[onGroupInfoChanged](../../callback/onGroupInfoChanged)  
:::
