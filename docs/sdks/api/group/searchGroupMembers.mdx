# searchGroupMembers

## 功能介绍

:::info 说明

在用户 ID 或群昵称字段中，用关键词搜索某个群的群成员

:::

:::caution 注意

（1）群成员才有搜索权限；  
（2）至少需指定一个搜索域；  
（3）多个域之间是或的关系；

:::

Future<List<GroupMembersInfo>> searchGroupMembers({

    List<String> keywordList = const [],

IMSDK.searchGroupMembers({

| 参数名称               | 参数类型 | 是否必填 | 描述                                           |

IMSDK.asyncApi('searchGroupMembers', operationID: string, {

    count: number;

| ---------------- | -------- | -------- | ------------------------------------------------------- |

| isSearchRemark   | boolean  | 是       | 是否以关键词搜索备注                                    |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                                 | 描述                   |
| --------------- | ------------------------------------------------------------------------ | ---------------------- |

| Promise.catch() | Promise<[CatchResponse](docs/sdks/class/response.mdx)\>                  | 调用失败回调           |
