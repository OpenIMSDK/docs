# changeGroupMemberMute

## 功能介绍

:::info 说明

群主或群管理员修改群成员禁言状态。

:::

:::caution 注意


群主可以禁言管理员和普通成员，管理员可以禁言普通成员。

**相关回调**:    
[onGroupApplicationAccepted](../../callback/onGroupMemberInfoChanged)  



:::

### 函数原型

```dart showLineNumbers
Future<dynamic> changeGroupMemberMute({
    required String groupID,
    required String userID,
    int seconds = 0,
    String? operationID,
  })
```

### 输入参数

| 参数名称 | 参数类型 | 是否必填 | 描述                    |
| -------- | -------- | -------- | ----------------------- |
| groupID  | String   | 是       | 群ID                   |
| userID   | String   | 是       | 被禁言的用户 ID         |
| seconds  | int      | 是       | 要禁言的时长，单位秒。为 0 时取消禁言 |

### 返回结果

| 参数名称 | 参数类型 | 描述                 |
| -------- | -------- | -------------------- |
| ~        | ~        | 无异常抛出则设置成功 |

```dart showLineNumbers

      groupID: 'groupID',
      userID: 'userID',
      seconds: 60,
    );
    // todo
```
