# setGroupMemberInfo

## 功能介绍

:::info 说明

修改群成员信息

:::

:::caution 注意

群主可以修改管理员和普通成员信息；管理员可以修改普通成员信息；  
群成员会收到 OnGroupMemberInfoChanged 回调；

:::

IMSDK.setGroupMemberInfo({

| 参数名称 | 参数类型 | 是否必填 | 描述           |

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('setGroupMemberInfo', operationID: string, {
  groupID: string;
  userID: string;
  ex: string;
}): Promise<void>
```

### 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述                                                    |
| ----------- | -------- | -------- | ------------------------------------------------------- |
| operationID | string   | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |
| groupID     | string   | 是       | 群 ID                                                   |
| userID      | string   | 是       | 用户 id                                                 |
| ex          | string   | 是       | 群成员扩展信息                                          |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                | 描述         |
| --------------- | ------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<void\>                                          | 调用成功回调 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/class/response.mdx)\> | 调用失败回调 |

### 代码示例

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

IMSDK.asyncApi('setGroupMemberInfo', Date.now(), {
  groupID: '',
  userID: '',
  ex: '',
})
  .then(() => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
