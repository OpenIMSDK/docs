# setConversationPrivateChat

## 功能介绍

:::info 说明

打开或者关闭"阅后即焚"，开启后发送的消息体上[AttachedInfoElem](docs/sdks/class/message/attachedInfoElem.mdx)的`isPrivateChat`为true，客户端可以根据这个标识位做阅后即焚的逻辑。

:::

:::caution 注意

**相关回调**：      
[onConversationChanged](docs/sdks/callback/onConversationChanged.mdx)

:::
