# 三方相关

## 模块概览

:::info

三方相关（文件、日志上传，firebase的token更新等）

:::

| 模块名称                                                                  | 模块功能简介             |
| ------------------------------------------------------------------------- | ------------------------ |
| [uploadLogs](docs/sdks/api/third/uploadLogs.mdx)                 | 客户端上传日志             |
| [logs](docs/sdks/api/third/logs.mdx)                 | 打印日志到SDK中           |
| [uploadFile](docs/sdks/api/third/uploadFile.mdx)                 | 独立上传文件到服务器  |
| [updateFcmToken](docs/sdks/api/third/updateFcmToken.mdx)                 | 设置或刷新firebase用户token  |
| [setAppBadge](docs/sdks/api/third/setAppBadge.mdx)                                     | 设置App角标使用,目前仅用于firebase
