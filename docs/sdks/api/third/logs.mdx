# logs


## 功能介绍

:::info 说明

该接口用于打印应用外部的日志信息，方便SDK统一收集上传服务器用于问题分析定位。


:::
:::caution 注意

该接口待更新。

:::

### 函数原型

```dart showLineNumbers
   Future logs({
```

| line | int   | 否       | 记录行数         |

| keyAndValues | List   | 否       | 参数         |

无

<TabItem value="uni-app">

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('getSelfUserInfo', operationID: string): Promise<SelfUserInfo>
```

### 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述                                                    |
| ----------- | -------- | -------- | ------------------------------------------------------- |
| operationID | string   | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                    | 描述         |
| --------------- | ----------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<[SelfUserInfo](docs/sdks/class/user/userInfo.mdx)\> | 个人信息对象 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/class/response.mdx)\>     | 调用失败回调 |

### 代码示例

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

IMSDK.asyncApi('getSelfUserInfo', IMSDK.uuid())
  .then((data) => {
    // data: 当前登录用户的个人信息
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
