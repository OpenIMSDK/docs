# onFriendAdded

## 功能介绍

:::info

两个用户成功建立好友关系后双方都会收到该回调。

:::

```js showLineNumbers
import { getSDK, CbEvents } from 'open-im-sdk-wasm';
// or
// import { OpenIMSDK, CbEvents } from 'open-im-sdk';

IMSDK.on(CbEvents.OnFriendAdded, ({ data }) => {
  // data 好友信息
});
```
