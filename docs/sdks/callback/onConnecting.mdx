:::info

:::

<TabItem value="Web">

### 返回原型

```ts showLineNumbers

onConnecting(data: WSEvent): void;

```

### 调用示例

```js showLineNumbers
import { getSDK, CbEvents } from 'open-im-sdk-wasm';
// or
// import { OpenIMSDK, CbEvents } from 'open-im-sdk';
// const IMSDK = new OpenIMSDK();
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnConnecting, () => {});
```

| 名称 | 类型                                    | 描述 |
| ---- | --------------------------------------- | ---- |
| data | [WSEvent](docs/sdks/class/response.mdx) | -    |
