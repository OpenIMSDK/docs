### 返回原型


### 返回结果

| 名称 | 类型                                                                                      | 描述     |
| ---- | ----------------------------------------------------------------------------------------- | -------- |
| data | [WSEvent](docs/sdks/class/response.mdx)<[GroupItem](docs/sdks/class/group/groupInfo.mdx)> | 群聊信息 |

### 调用示例

```js showLineNumbers
import { getSDK } from 'open-im-sdk-wasm';
import { CbEvents } from 'open-im-sdk-wasm/constant';
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnGroupDismissed, ({ data }) => {
  // data 群信息
});
```
