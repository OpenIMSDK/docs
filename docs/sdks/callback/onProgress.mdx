# onProgress

## 功能介绍

:::info

消息发送进度回调，一般在发送文件类消息时使用。

:::

<TabItem value="Flutter">

### 返回原型

```dart showLineNumbers

Function(String clientMsgID, int progress)? onProgress;

```

### 返回结果

| 名称   | 类型      | 描述     |
| ------ | --------- | -------- |
| clientMsgID | String | 消息ID |
| progress | int | 进度值 |

</TabItem>

<TabItem value="iOS">

### 返回原型

```swift showLineNumbers

typedef void (^OIMNumberCallback)(NSInteger number);

```

### 返回结果

| 名称   | 类型      | 描述     |
| ------ | --------- | -------- |
| number | NSInteger | 进度值 |

</TabItem>

<TabItem value="Android">
### 返回原型

```java showLineNumbers

 void onProgress(long progress)

```
### 返回结果

| 名称   | 类型      | 描述     |
| ------ | --------- | -------- |
| progress | long | 发送消息进度 |

</TabItem>

<TabItem value="Web">

### 发送消息进度回调

```ts showLineNumbers

onProgress(data: WSEvent<number>): void;

```

### 调用示例

```js showLineNumbers
import { getSDK, CbEvents } from 'open-im-sdk-wasm';
// or
// import { OpenIMSDK, CbEvents } from 'open-im-sdk';
// const IMSDK = new OpenIMSDK();
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnProgress, ({ data }) => {
  // data 消息发送进度
});
```

</TabItem>
