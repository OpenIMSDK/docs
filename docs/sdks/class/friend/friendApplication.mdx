# FriendApplication

## 功能介绍

:::info

好友信息，黑名单用户基本信息，注意黑名单是双向关系。

:::

### FriendApplicationInfo

| 字段名称      | 字段类型 | 描述                                |
| ------------- | -------- | ----------------------------------- |
| fromUserID    | String   | 发起者 UserID                       |
| fromNickname  | String   | 发起者昵称                          |
| fromFaceURL   | String   | 发起者头像链接                      |
| fromGender    | String   | 发起者性别                          |
| toUserID      | String   | 被添加者 UserID                     |
| toNickname    | String   | 被添加者昵称                        |
| toFaceURL     | String   | 被添加者头像链接                    |
| toGender      | String   | 被添加者性别                        |
| handleResult  | int      | 处理结果（1 同意/-1 拒绝/0 未处理） |
| reqMsg        | String   | 申请添加好友时附带的消息            |
| createTime    | int      | 创建时间                            |
| handlerUserID | String   | 处理者 UserID                       |
| handleMsg     | String   | 处理好友申请时附带的消息            |
| handleTime    | int      | 处理时间                            |
| ex            | String   | 应用层扩展字段                      |
| attachedInfo  | String   | SDK 内部扩展字段，应用层无需关注    |

</TabItem>
