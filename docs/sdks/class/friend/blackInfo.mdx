# BlackInfo

## 功能介绍

:::info

黑名单信息，黑名单用户基本信息，注意黑名单是双向关系。

:::

### BlacklistInfo

| 字段名称       | 字段类型 | 描述                             |
| -------------- | -------- | -------------------------------- |
| userID         | String   | 用户 id                          |
| nickname       | String   | 黑名单的昵称                     |
| faceURL        | String   | 头像链接                         |
| gender         | int      | 性别                             |
| createTime     | int      | 拉黑时间                         |
| addSource      | int      | 拉黑方式                         |
| operatorUserID | String   | 使之成为黑名单的 UesrID          |
| attachedInfo   | String   | SDK 内部扩展字段，应用层无需关注 |
| ex             | String   | 应用层扩展字段                   |

</TabItem>
