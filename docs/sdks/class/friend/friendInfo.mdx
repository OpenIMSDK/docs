# FriendInfo

## 功能介绍

:::info

好友信息，黑名单用户基本信息，注意黑名单是双向关系。

:::

### FriendInfo

| 字段名称       | 字段类型 | 描述                             |
| -------------- | -------- | -------------------------------- |
| userID         | String   | 用户 id                          |
| nickname       | String   | 好友的昵称                       |
| faceURL        | String   | 头像链接                         |
| gender         | int      | 性别                             |
| phoneNumber    | String   | 手机号                           |
| birthTime      | String   | 出生日期                         |
| birth          | int      | 出生日期                         |
| email          | String   | 邮箱                             |
| remark         | String   | 好友备注                         |
| createTime     | int      | 成为好友时间                     |
| addSource      | int      | 成为好友的方式                   |
| operatorUserID | String   | 使双方成为好友的 UesrID          |
| attachedInfo   | String   | SDK 内部扩展字段，应用层无需关注 |
| ex             | String   | 应用层扩展字段                   |

</TabItem>
