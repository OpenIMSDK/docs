# PublicInfo

## 功能介绍

:::info

用户公开信息，主要是基本信息，不包括手机等其它隐私字段

:::

### PublicUserInfo

| 字段名称 | 字段类型 | 描述                        |
| -------- | -------- | --------------------------- |
| userID   | String   | 用户 ID，必须保证 IM 内唯一 |
| nickname | String   | 昵称                        |
| faceURL  | String   | 头像链接                    |
| gender   | int      | 性别                        |
| ex       | String   | 用户扩展字段                |

</TabItem>
