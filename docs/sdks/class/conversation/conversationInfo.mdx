### ConversationInfo

| 字段名称          | 字段类型                                           | 描述                                                                  |
| ----------------- | -------------------------------------------------- | --------------------------------------------------------------------- |
| conversationID    | String                                             | 会话 ID                                                               |
| conversationType  | int                                                | 会话类型 [ConversationType](docs/sdks/enum/conversationType.mdx)      |
| userID            | String                                             | 用户 ID                                                               |
| groupID           | String                                             | 群 ID                                                                 |
| showName          | String                                             | 展示名称                                                              |
| faceURL           | String                                             | 头像地址                                                              |
| recvMsgOpt        | int                                                | 接收消息 免打扰 0：正常；1：不接受消息；2：接受在线消息不接受离线消息 |
| unreadCount       | int                                                | 未读数                                                                |
| groupAtType       | int                                                | @类型 [GroupAtType](docs/sdks/enum/groupAtType.mdx)                   |
| latestMsgSendTime | int                                                | 最近消息发送时间                                                      |
| draftText         | String                                             | 草稿                                                                  |
| draftTextTime     | int                                                | 生成草稿时间                                                          |
| isPinned          | bool                                               | 是否置顶                                                              |
| isPrivateChat     | bool                                               | 是否开启了私聊                                                        |
| burnDuration      | int                                                | 私聊时长                                                              |
| isNotInGroup      | bool                                               | 是否还在组内，如果退群返回 true                                       |
| isPrivateChat     | bool                                               | 是否开启了私聊                                                        |
| attachedInfo      | String                                             | 附加信息                                                              |
| latestMsg         | [Message](docs/sdks/class/message/messageInfo.mdx) | 最近的消息                                                            |
| ex                | String                                             | 应用层扩展字段                                                        |

</TabItem>
