# 我们的使命

从 **OpenIM** 创立之日起，我们始终坚信“开源”为核心驱动。这不仅体现了互联网的自由、平等、分享的理念，也回应了全球的数据安全挑战。

数据泄露、勒索软件和隐私侵犯等问题不断警示我们安全的重要性。为此，我们提供了一个既低成本又安全的IM服务，以满足企业对数据保护的需求。

> ⭐️ 我们感谢每一位给我们点赞的开发者。您的支持是我们追求成为IM开源第一的基石！

[🔗 点击查看我们的Github仓库](https://github.com/OpenIMSDK/Open-IM-Server)

# 我们的团队

由 **IM/RTC** 资深架构师组成的我们，始终坚信开源技术可以为用户创造真实价值。我们欢迎全球的技术爱好者加入我们，与我们一起优化技术，让每一个应用都能拥有IM功能。
