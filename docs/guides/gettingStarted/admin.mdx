# 管理后台 🖥️
### 🟢 开放端口

| TCP 端口  | 说明                    | 操作 ⚙️                                      |
|:--------:|:-----------------------:|:--------------------------------------------:|
| TCP:11002| `http://ip:11002` 访问管理后台 | 端口放行或 nginx 反向代理，并关闭防火墙 |

## 📌 访问管理后台

:::tip
在浏览器中输入 `http://ip:11002` 来访问管理后台。此 ip 为服务端 ip ，且确保您的浏览器能访问。默认账号和密码均为admin1
