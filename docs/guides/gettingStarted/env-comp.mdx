## 🌟 环境要求

| **注意事项** | **详细说明**                                                  |
|------------|------------------------------------------------------------|
| 操作系统     | linux系统，Ubuntu 22.04最优，root用户    |
| 硬件资源     | 至少4G可用内存，对于用户量<10万推荐单机部署 (8C 16G 10Mb 1T)  |
| Golang      | v1.19 或更高                                                   |
| Docker      | v24.0.5 或更高                                                 |
| Git         | v2.17.1 或更高                                                 |

## 📦 存储组件要求

| **存储组件** | **建议版本**  |
|:------------:|:-------------:|
| MongoDB      | v6.0.2+       |
| Redis        | v7.0.0+       |
| Zookeeper    | v3.8        |
| Kafka        | 3.5.1        |
| MySQL        | v5.7+         |
| MinIO        | 最新版        |
