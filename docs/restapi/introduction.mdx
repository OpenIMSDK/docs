# OpenIMServer APIs

## 🚀 REST API

OpenIMServer 提供了 **REST API** 供业务系统使用，以增强业务功能。如：

- 🛠 建立群组
- 📬 推送消息

更多功能参考后面的API文档

> ⚠️ **注意**: 调用 **REST API** ，需要以 **APP 管理员** 身份进行，此服务由端口 `10002` 提供。所谓 **APP 管理员**，即有超级权限的用户。OpenIMServer 内置了一个 APP管理员，它的userID 为imAdmin


在开始调用 API 之前，需首先通过 [获取token](./apis/authenticationManagement/getUserToken)使用secret获取 APP 管理员的 token。
此外，对于[用户注册](./apis/userManagement/userRegister) 接口，也需要使用secret验证。

## 🔗 Webhooks

OpenIMServer 还提供了强大的回调功能，以满足更多的业务需求。**回调** 的定义是，当某事件发生前或后，OpenIMServer 会向业务服务器发出请求。如：

- 📤 发送消息前的回调
- 📥 发送消息后的回调
