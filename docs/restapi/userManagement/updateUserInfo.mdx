## 更新用户信息

</center>

### 简要描述
- 更新用户在IM中的资料，主要是头像、昵称
#### 接口状态
- 已完成
### 请求方式
- `post`

### 请求参数示例


```json
{
  "userInfo": {
    "userID": "11111112",
    "nickname": "alantestuid3",
    "faceURL": "1111111"
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|userInfo|必填|object|用户信息|
|userInfo.userID|必填|string|用户lD|
|userInfo.nickname|必填|string|用户名|
|userInfo.faceURL|必填|string|用户头像|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
