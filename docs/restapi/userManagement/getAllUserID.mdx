## 分页获取所有用户ID

</center>

### 简要描述
- 获取所有已注册的用户ID
### 请求方式
- `post` 
### 请求URL
- `{{host}}/user/get_all_users_uid`

### 请求参数示例


```json
{
  "pagination": {
    "pageNumber": 2,
    "showNumber": 2
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|pagination|必填|object|分页结构体|
|pagination.pageNumber|必填|string|分页号，从1开始|
|pagination.showNumber|必填|string|每个分页显示用户数|
### 成功返回示例

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|total|string|用户总数|
|userIDs|array|需要查询的用户ID数组|
