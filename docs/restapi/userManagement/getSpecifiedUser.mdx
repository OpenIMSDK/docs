## 获取指定用户列表

</center>

### 简要描述
- 获取已注册用户列表
### 请求方式
- `post` 
### 请求URL
- `{{host}}/user/get_users_info`

### 请求参数示例


```json
{
  "userIDs": [
    "11111111",
    "11111112"
  ]
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|userIDs|必填|array|需要查询的用户ID数组|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|usersInfo|array|用户信息数组|
|usersInfo.userID|string|用户ID|
|usersInfo.nickname|string|用户名|
|usersInfo.faceURL|string|用户头像|
|usersInfo.ex|string|扩展字段|
|usersInfo.createTime|string|创建时间|
|usersInfo.appMangerLevel|string|管理员等级|
|usersInfo.globalRecvMsgOpt|string|全局是否接收消息，0为接收，1为不接收|
### 失败返回示例


```json
{
  "errCode": 1001,
  "errMsg": "ArgsError",
  "errDlt": ""
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
