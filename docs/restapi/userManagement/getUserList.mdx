## 分页获取已注册用户列表

</center>

### 简要描述
- 获取已注册用户列表，支持分页拉取
### 请求方式
- `post` 
### 请求URL
- `{{host}}/user/get_users`

### 请求参数示例


```json
{
  "pagination": {
    "pageNumber": 2,
    "showNumber": 2
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|pagination|必填|object|分页结构体|
|pagination.pageNumber|必填|string|分页号|
|pagination.showNumber|必填|string|每个分页显示用户数|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|total|string|用户总数|
|users|array|用户列表|
|users.userID|string|用户ID|
|users.nickname|string|用户名|
|users.faceURL|string|用户头像|
|users.ex|string|拓展字段|
|users.createTime|string|创建时间|
|users.appMangerLevel|string|管理员等级|
|users.globalRecvMsgOpt|number|全局是否接收消息，0为接收，1为不接收|
### 失败返回示例


```json

  "errCode": 500,
  "errMsg": "ServerInternalError",
  "errDlt": "runtime error: invalid memory address or nil pointer dereference"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
