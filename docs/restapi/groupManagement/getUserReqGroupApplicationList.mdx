## 获取用户自己的主动加群申请

</center>

### 简要描述
- 获取用户自己的主动加群申请
### 请求方式
- `post` 
### 请求URL
- `{{host}}/group/get_user_req_group_applicationList`

### 请求参数示例


```json
{
  "userID": "1225441072",
  "pagination": {
    "showNumber": 10,
    "pageNumber": 1
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|userID|必填|string|用户ID|
|pagination|必填|object|分页结构体|
|pagination.showNumber|选填|string|当前页数，从1开始|
|pagination.pageNumber|选填|string|每页展示加群申请数|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|total|string|加群申请总数|
|groupRequests|array|加群申请|
|groupRequests.userInfo|object|加群的用户信息|
|groupRequests.userInfo.userID|string|加群的用户ID|
|groupRequests.userInfo.nickname|string|加群的昵称|
|groupRequests.userInfo.faceURL|string|加群的头像|
|groupRequests.userInfo.ex|string|群的拓展字段|
|groupRequests.groupInfo|object|群组信息|
|groupRequests.groupInfo.groupID|string|群ID|
|groupRequests.groupInfo.groupName|string|群名|
|groupRequests.groupInfo.notification|string|群通知|
|groupRequests.groupInfo.introduction|string|群介绍|
|groupRequests.groupInfo.faceURL|string|群头像|
|groupRequests.groupInfo.ownerUserID|string|群主ID|
|groupRequests.groupInfo.createTime|int|群创建的时间|
|groupRequests.groupInfo.memberCount|string|群成员数量|
|groupRequests.groupInfo.ex|string|群拓展字段|
|groupRequests.groupInfo.status|int|群状态|
|groupRequests.groupInfo.creatorUserID|string|创建者ID|
|groupRequests.groupInfo.groupType|int|群类型，当前只会为2|
|groupRequests.groupInfo.needVerification|int|是否需要验证|
|groupRequests.groupInfo.lookMemberInfo|int|是否能查看群成员信息|
|groupRequests.groupInfo.applyMemberFriend|int|是否能加其他群成语为好友|
|groupRequests.groupInfo.notificationUpdateTime|int|群通知更新时间|
|groupRequests.groupInfo.notificationUserID|string|群通知用户ID|
|groupRequests.handleResult|string|处理结果，0为未处理，1为已处理，2为已拒绝|
|groupRequests.reqMsg|string|申请加群的消息|
|groupRequests.handleMsg|string|处理的信息|
|groupRequests.reqTime|string|申请时间|
|groupRequests.handleUserID|string|处理的用户ID|
|groupRequests.handleTime|string|处理的时间|
|groupRequests.ex|string|群申请拓展字段|
|groupRequests.joinSource|string|加群来源|
|groupRequests.inviterUserID|string|被邀请者的ID|
