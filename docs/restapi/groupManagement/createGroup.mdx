### 请求参数示例


```json
{
  "memberUserIDs": [
    "1225441072"
  ],
  "adminUserIDs": [
    "2065939257"
  ],
  "ownerUserID": "1054527962",
  "groupInfo": {
    "groupID": "xadxwr24",
    "groupName": "yourg group name",
    "notification": "notification",
    "introduction": "introduction",
    "faceURL": "faceURL url",
    "ex": "ex",
    "groupType": 0,
    "needVerification": 0,
    "lookMemberInfo": 0,
    "applyMemberFriend": 0
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|memberUserIDs|必填|string|初始化群成员ID列表|
|adminUserIDs|必填|array|管理用户用户ID列表|
|ownerUserID|必填|string|群主ID|
|groupInfo|必填|object|0为普通群，2为超级群|
|groupInfo.groupID|选填|string|群ID|
|groupInfo.groupName|必填|string|群名称|
|groupInfo.notification|必填|string|群通知|
|groupInfo.introduction|必填|string|群介绍|
|groupInfo.faceURL|必填|string|群头像|
|groupInfo.ex|必填|string|群拓展字段|
|groupInfo.groupType|必填|number|群类型|
|groupInfo.needVerification|必填|string|进群是否需要验证|
|groupInfo.lookMemberInfo|必填|string|是否能查看其他群成员信息|
|groupInfo.applyMemberFriend|必填|string|群成员是否能添加好友|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|groupInfo.groupID|string|群ID|
|groupInfo.groupName|string|群名称|
|groupInfo.notification|string|群通知|
|groupInfo.introduction|string|群介绍|
|groupInfo.faceURL|string|群头像|
|groupInfo.ownerUserID|string|群主ID|
|groupInfo.createTime|string|创建时间|
|groupInfo.memberCount|number|群成员数量|
|groupInfo.ex|string|群拓展字段|
|groupInfo.status|string|群状态|
|groupInfo.creatorUserID|string|群创建者ID|
|groupInfo.groupType|number|群类型|
|groupInfo.needVerification|string|进群是否需要验证|
|groupInfo.lookMemberInfo|string|是否能查看其他群成员信息|
|groupInfo.applyMemberFriend|string|群成员是否能添加好友|
|groupInfo.notificationUpdateTime|string|群通知更新时间|
|groupInfo.notificationUserID|string|群通知用户|
### 失败返回示例


```json
{
  "errCode": 1202,
  "errMsg": "GroupIDExisted",
  "errDlt": "group id existed xadxwr24: [1202]GroupIDExisted"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
