### 请求参数示例


```json
{
  "groupID": "xadxwr24",
  "fromUserID": "2966378840",
  "handledMsg": "",
  "handleResult": 1
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|groupID|必填|string|群ID|
|fromUserID|必填|string|申请用户ID|
|handledMsg|选填|string|处理信息|
|handleResult|必填|string|处理结果 -1为拒绝该用户加群，1为同意用户加群|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
