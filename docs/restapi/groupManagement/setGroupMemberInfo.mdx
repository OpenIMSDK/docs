### 请求参数示例


```json
{
  "members": [
    { 
      "groupID": "2137448827",
      "userID": "2699373280",
      "nickName": "newName",
      "faceURL": "new faceURL",
      "roleLevel": 60,
       "ex":""
    }
  ]
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|members|必填|array|要修改的群成员对象|
|members.groupID|必填|string|群ID|
|members.userID|必填|string|群成员用户ID|
|members.nickName|选填|string|群成员昵称|
|members.faceURL|选填|string|群成员头像|
|members.roleLevel|选填|string|群成员等级，1为普通，2为群主，3为管理员|
|members.ex|选填|string|群成员拓展字段|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
