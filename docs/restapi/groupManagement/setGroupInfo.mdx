### 请求参数示例


```json
{
  "groupInfoForSet": {
    "groupID": "xadxwr24",
    "groupName": "new name",
    "notification": "new notification",
    "introduction": "new introduction",
    "faceURL": "www.newfaceURL.com",
    "ex": "new ex",
    "needVerification": 1,
    "lookMemberInfo": 1,
    "applyMemberFriend": 1
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|groupInfoForSet|必填|object||
|groupInfoForSet.groupID|必填|string|要设置的群ID|
|groupInfoForSet.groupName|选填|string|要设置的群名|
|groupInfoForSet.notification|选填|string|新群通知|
|groupInfoForSet.introduction|选填|string|新群介绍|
|groupInfoForSet.faceURL|选填|string|新群头像|
|groupInfoForSet.ex|选填|string|新群拓展字段|
|groupInfoForSet.needVerification|选填|string|0为申请需要同意 邀请直接进，1为所有人进群需要验证，除了群主管理员邀请进群，2为直接进群|
|groupInfoForSet.lookMemberInfo|选填|string|0为用户开源查看其他用户信息，1为不能查看|
|groupInfoForSet.applyMemberFriend|选填|string|0为群成员可以加其他用户好友，1为不能|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
