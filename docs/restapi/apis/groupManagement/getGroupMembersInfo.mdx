## 获取指定群成员

</center>

### 简要描述
- 获取指定群成员的详细信息。
### 请求方式
- `post` 
### 请求URL
- `{API_ADDRESS}/group/get_group_members_info` 


### Header
|header名|示例值|选填|类型|说明|
|:----    |:-------    |:--- |---|------      |
|operationID|1646445464564|必填|string|用于全局链路追踪，建议使用时间戳，在每个请求中独立|
|token|eyJhbxxxx3Xs|必填|string|管理员token|


### 请求参数示例


```json
{
  "groupID": "324253771",
  "userIDs": ["7009965934"]
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|groupID|必填|string|群ID|
|userIDs|必填|array|群成员ID列表|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息，为空|
|errDlt|errDlt|错误详细信息，为空|
|data|object|通用数据对象，具体结构见下方|
|members|array|[群成员信息](docs/restapi/commonFields.mdx#groupmemberinfo)列表|
### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |

|errMsg|string|错误简要信息|
