## 修改系统通知号信息

</center>

### 简要描述

- 修改系统通知号信息，包括头像和名称。

### 请求方式

- `post`

### 请求 URL

- `{API_ADDRESS}/user/update_notification_account`

### Header

| header 名   | 示例值        | 选填 | 类型   | 说明                         |
| :---------- | :------------ | :--- | ------ | ---------------------------- |
| operationID | 1646445464564 | 必填 | string | 用于全局链路追踪，建议使用时间戳，在每个请求中独立 |
| token       | eyJhbxxxx3Xs  | 必填 | string | 管理员 token                 |

### 请求参数示例

```json
{
  "userID": "1974356875",
  "nickName": "notification1",
  "faceURL": "url"
}
```

| 字段名   | 选填 | 类型   | 说明           |
| :------- | :--- | :----- | -------------- |
| userID   | 必填 | string | 系统通知号 ID  |
| nickName | 必填 | string | 系统通知号名称 |
| faceURL  | 必填 | string | 系统通知号头像 |

### 成功返回示例

```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```

### 成功返回示例的参数说明

| 参数名  | 类型   | 说明                      |
| :------ | :----- | :------------------------ |
| errCode | int    | 错误码,0 表示成功         |
| errMsg  | string | 错误简要信息，为空 |
| errDlt  | errDlt | 错误详细信息，为空 |

### 失败返回示例

```json
{
  "errCode": 1001,
  "errMsg": "ArgsError",
  "errDlt": "1001 ArgsError"
}
```

### 失败返回示例的参数说明

| 参数名  | 类型   | 说明                          |
| :------ | :----- | :---------------------------- |
| errCode | int    | 错误码,具体查看全局错误码文档 |
| errMsg  | string | 错误简要信息                  |
| errDlt  | errDlt | 错误详细信息                  |
