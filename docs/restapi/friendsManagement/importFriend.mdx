## 导入好友

</center>

### 简要描述
- 导入好友关系，该接口需要使用管理员token
### 请求方式
- `post` 
### 请求URL
- `{{host}}/friend/import_friend`

### 请求参数示例


```json
{
  "ownerUserID": "2712330015",
  "friendUserIDs": ["3437150835", "5225385877"]
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|ownerUserID|必填|string|需要导入好友的用户|
|friendUserIDs|必填|string|导入的好友用户ID列表|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
### 失败返回示例


```json
{
  "errCode": 1003,
  "errMsg": "DuplicateKeyError",
  "errDlt": ": [1003]DuplicateKeyError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
