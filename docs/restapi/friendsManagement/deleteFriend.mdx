### 请求参数示例


```json
{
  "ownerUserID":"11111111",
  "friendUserID":"11111113"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|ownerUserID|必填|string|用户ID|
|friendUserID|必填|string|被删除的用户ID|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|errCode|int|错误码,0表示成功|

```json

  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": "[1004]RecordNotFoundError"
```
