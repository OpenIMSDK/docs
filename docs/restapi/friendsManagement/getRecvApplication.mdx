## 获取收到的好友申请列表

</center>

### 简要描述
- 获取收到的好友申请列表，支持分页拉取
### 请求方式
- `post` 
### 请求URL
- `{{host}}/friend/get_friend_apply_list`

### 请求参数示例


```json
{
  "userID": "11111112",
  "pagination": {
    "pageNumber": 1,
    "showNumber": 1
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|

|pagination.pageNumber|必填|string|分页号|
|pagination.showNumber|必填|string|每个分页显示用户数|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|FriendRequests|array|收到的的好友申请列表|
|FriendRequests.fromUserID|string|申请好友的用户ID|
|FriendRequests.fromNickname|string|申请好友的用户名|
|FriendRequests.fromFaceURL|string|申请好友的头像|
|FriendRequests.toUserID|string|被申请的用户ID|
|FriendRequests.toNickname|string|被申请的用户名|
|FriendRequests.toFaceURL|string|被申请的用户头像|
|FriendRequests.handleResult|string|申请的结果，0代表未处理，-1代表拒绝|
|FriendRequests.reqMsg|string|申请的信息|
|FriendRequests.createTime|int|申请时间（毫秒时间戳）|
|FriendRequests.handlerUserID|string|处理用户的ID|
|FriendRequests.handleMsg|string|处理的信息|
|FriendRequests.handleTime|int|处理的时间（毫秒时间戳）|
|FriendRequests.ex|string|用户使用拓展字段|
|total|string|收到的好友申请总数|
### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": "[1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
