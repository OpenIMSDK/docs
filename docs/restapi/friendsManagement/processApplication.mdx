## 好友申请处理

</center>

### 简要描述
- 用户同意/拒绝加好友申请
### 请求方式
- `post` 
### 请求URL
- `{{host}}/friend/add_friend_response`

### 请求参数示例


```json
{
  "fromUserID": "11111111",
  "toUserID": "11111112",
  "handleResult": 1,
  "handleMsg": "agree"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|fromUserID|必填|string|主动发起的申请者用户ID|
|toUserID|必填|string|被添加好友的用户ID|
|handleResult|必填|string|1为同意申请，-1为拒绝申请|
|handleMsg|必填|string|处理的信息|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": "==&gt; github.com/OpenIMSDK/Open-IM-Server/pkg/common/db/relation.(*FriendRequestGorm).Take()@52: : record not found"

```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
