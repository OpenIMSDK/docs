</center>

### 简要描述

### 请求方式
- `post` 
### 请求URL
- `{{host}}/friend/is_friend`

### 请求参数示例


```json
{
  "userID1": "11111111",
  "userID2": "11111112"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|userID1|必填|string|第一个用户ID|
|userID2|必填|string|第二个用户ID|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {
    "inUser1Friends": true,
    "inUser2Friends": true
  }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|inUser1Friends|boolean|userID2是否在userID1的好友列表中，为true表示在好友列表中，为false表示不在好友列表中|
|inUser2Friends|boolean|userID1是否在userID2的好友列表中，为true表示在好友列表中，为false表示不在好友列表中|
