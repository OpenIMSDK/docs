### 请求参数示例


```json
{
  "status": 0,
  "userIDs": [],
  "codes": [],
  "keyword": "666",
  "pagination": {
    "pageNumber": 1,
    "showNumber": 10
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|status|必填|string|0:全部,1:已使用,2:未使用|
|userIDs|必填|string|用户ID数组|
|codes|必填|string|搜索的邀请码|
|keyword|必填|string|搜索关键字|
|pagination|必填|object|分页|
|pagination.pageNumber|必填|int|分页号|
|pagination.showNumber|必填|int|每个分页展示邀请码数量|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {
    "total": 1,
    "list": [
      {
        "invitationCode": "666",
        "createTime": 1690188678847,
        "usedUserID": "",
        "usedUser": null
      }
    ]
  }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|total|string||
|list|array||
|list.invitationCode|string||
|list.createTime|string||
|list.usedUserID|string||
|list.usedUser|object||
