### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|total|string|总数|
|groups|array||
|groups.groupID|string|群组ID|
|groups.groupName|string|群名称|
|groups.notification|string|通知|
|groups.introduction|string|介绍|
|groups.faceURL|string|头像|
|groups.ownerUserID|string|群主ID|
|groups.createTime|string|创建时间|
|groups.memberCount|string|成员数量|
|groups.ex|string|扩展字段|
|groups.status|string|群状态|
|groups.creatorUserID|string|创建者ID|
|groups.groupType|int|群类型|
|groups.needVerification|string|入群是否需要验证|
|groups.lookMemberInfo|string|是否能查看其他群成员信息|
|groups.applyMemberFriend|string|群成员是否能添加好友|
|groups.notificationUpdateTime|string|群通知更新时间|
|groups.notificationUserID|string|群通知发布者ID|
