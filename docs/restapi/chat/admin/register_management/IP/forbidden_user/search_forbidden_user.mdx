### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|total|string|限制总数|
|limits|array|限制|
|limits.userID|string|用户ID|
|limits.ip|string|ip，限制用户在此ip登录|
|limits.createTime|string|创建时间|
|limits.user|object|用户|
|limits.user.userID|string|用户ID|
|limits.user.account|string|用户账户|
|limits.user.email|string|用户邮箱|
|limits.user.nickname|string|用户名|
|limits.user.faceURL|string|头像|
|limits.user.gender|string|性别|
|limits.user.level|string|管理员等级|
