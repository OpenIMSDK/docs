### 简要描述
- 搜索禁止注册登录IP，支持分页拉取
### 请求方式
- `post` 
### 请求URL
- `{{admin_addr}}/forbidden/ip/search`

### 请求参数示例


```json
{
  "status": 0,
  "keyword": "",
  "pagination": {
    "pageNumber": 0,
    "showNumber": 10
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|status|必填|string|0:不做筛选,1:都不限制,2:只限制注册,3:只限制登录,4:限制注册,5:限制登录,6:都限制|
|keyword|必填|string|搜索关键词|
|pagination|必填|object|分页|
|pagination.pageNumber|必填|string|分页号|
|pagination.showNumber|必填|string|每个分页展示ip数|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {
    "total": 1,
    "forbiddens": [
      {
        "ip": "127.0.0.1",
        "limitRegister": true,
        "limitLogin": true,
        "createTime": 1688550488908
      }
    ]
  }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|total|string|封禁总数|
|forbiddens|array|封禁|
|forbiddens.ip|string|封禁ip|
|forbiddens.limitRegister|string|禁止注册|
|forbiddens.limitLogin|string|禁止登录|
|forbiddens.createTime|string|创建时间|
