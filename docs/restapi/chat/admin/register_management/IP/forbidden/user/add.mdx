### 请求参数示例


```json
{
  "limits": [
    {
      "ip": "127.0.0.1",
      "userID": "6646295080"
    }
  ]
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|limits|必填|array|限制数组|
|limits.ip|必填|string|ip，限制用户在此ip登录|
|limits.userID|必填|string|用户ID|
