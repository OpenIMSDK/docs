### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|account|string|账户|
|password|string|密码|
|faceURL|string|头像|
|nickname|string|用户名|
|userID|string|用户ID|
|level|string|管理员等级|
|createTime|string|创建时间|
