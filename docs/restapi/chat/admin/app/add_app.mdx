### 请求参数示例


```json
{
  "id": "12345",
  "name": "测试小程序",
  "appID": "__UNI__7460852111",
  "icon": "https://doc.rentsoft.cn/images/logo.png",
  "url": "http://203.56.175.233:51376/__UNI__7460850.wgt",
  "md5": "4d44d4e251c59e83a9a08882241a8841",
  "size": 213980,
  "version": "1.0.0",
  "status": 1
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|id|选填|string|为空服务端生成|
|name|必填|string|小程序名称|
|appID|必填|string|小程序ID|
|icon|必填|string|图标|
|url|必填|string|url地址|
|md5|必填|string||
|size|必填|string|大小|
|version|必填|string||
|status|必填|string||
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
