```json
{
  "keyword": "test",
  "pagination": {
    "pageNumber": 1,
    "showNumber": 10

```
|字段名|选填|类型|说明|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|total|string|总数|
|applets|array||
|applets.id|string|ID|
|applets.name|string|小程序名称|
|applets.appID|string|小程序ID|
|applets.icon|string|图标|
|applets.url|string|url地址|
|applets.md5|string|md5加密|
|applets.size|string|大小|
|applets.version|string|版本|
|applets.priority|string|优先级|
|applets.status|string|状态|
|applets.createTime|string|创建时间|
