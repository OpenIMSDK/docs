### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {
    "config": {
      "aaa": "bbb",
      "discoverPageURL": "12345",
      "key": "1111111"
    }
  }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|config|object|返回的是一个map,服务器没用配置会返回null|
|config.aaa|string|所有key对应的值都是字符串|
|config.discoverPageURL|string||
|config.key|string||
