### 请求参数示例


```json
{
  "userID": "9587519502",
  "nickname": "123456",
  "account": null,
  "phoneNumber": null,
  "areaCode": null,
  "email": null,
  "faceURL": "",
  "gender": 2,
  "level": null,
  "birth": 2,
  "allowAddFriend": 2,
  "allowBeep": 2,
  "allowVibration": 2,
  "globalRecvMsgOpt": 2
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|nickname|选填|string|昵称|
|faceURL|选填|string|头像|
|gender|选填|string|性别|
|allowAddFriend|选填|string|是否允许添加好友|
|allowBeep|选填|string|允许响铃|
|allowVibration|选填|string|允许震动|
|globalRecvMsgOpt|必填|string|全局接收消息设置|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
