### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|users|array||
|users.userID|string|用户ID|
|users.password|string|密码|
|users.account|string|账号|
|users.phoneNumber|string|手机号|
|users.areaCode|string|区号|
|users.email|string|邮箱|
|users.nickname|string|昵称|
|users.faceURL|string|头像|
|users.gender|string|性别|
|users.level|string|管理员等级|
|users.birth|string|生日|
|users.allowAddFriend|string|是否允许添加好友|
|users.allowBeep|string|允许响铃|
|users.allowVibration|string|允许震动|
|users.globalRecvMsgOpt|string|全局是否接收消息，0为接收，1为不接收|
