### 请求参数示例


```json
{
  "usedFor": 1,
  "invitationCode": "",
  "deviceID": "",
  "platform": 1,
  "areaCode": "+86",
  "phoneNumber": "13812345678"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|usedFor|必填|string|1:注册,2:重置密码,3登录|
|invitationCode|选填|string|邀请码|
|deviceID|选填|string|设备ID|
|platform|选填|string|平台|
|areaCode|必填|string|+86|
|phoneNumber|必填|string|手机号|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
