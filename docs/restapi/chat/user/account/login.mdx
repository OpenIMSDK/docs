```json
{
  "verifyCode": "666666",
  "deviceID": "",
  "platform": 1,
  "areaCode": "+86",
  "phoneNumber": "13944444444",
  "account": "",
  "password": ""
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|verifyCode|必填|string|验证码|

|platform|必填|string|平台ID|
|areaCode|必填|string|区号|

|account|必填|string|账号|
|password|必填|string|密码|
