### 请求参数示例


```json
{
  "invitationCode": "",
  "verifyCode": "666666",
  "deviceID": "",
  "platform": 1,
  "autoLogin": true,
  "user": {
    "nickname": "testname",
    "faceURL": "",
    "birth": 0,
    "gender": 1,
    "email": "",
    "account": "",
    "areaCode": "+86",
    "phoneNumber": "13963999332",
    "password": "888888888"
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|invitationCode|选填|string|邀请码|
|verifyCode|必填|string|验证码|
|deviceID|必填|string|设备ID|
|platform|必填|string|平台|
|autoLogin|选填|boolean|自动登录，创建token，创建失败不会返回错误，但token为空字符串|
|user|必填|object||
|user.nickname|必填|string|用户名|
|user.faceURL|必填|string|头像|
|user.birth|必填|string|生日|
|user.gender|必填|string|性别|
|user.email|选填|string|邮箱|
|user.account|选填|string|账号|
|user.areaCode|必填|string|区号|
|user.phoneNumber|必填|string|手机号|
|user.password|选填|string|密码|
