## III. Verification

### PC Web Verification

:::tip
In your browser, enter `http://ip:11001` to access the PC Web interface. This IP corresponds to the server's `OPENIM_IP`. Ensure your browser can access it. For first-time use, register with your mobile number; the verification code defaults to `666666`.
:::

### App Verification

Scan the QR code below or click [here](https://www.pgyer.com/IM-FCER) to download the app.

### Admin Panel Verification

In your browser, enter `http://ip:11002` to access the admin panel. This IP corresponds to the server's `OPENIM_IP`. Ensure your browser can access it. The default username and password are both `chatAdmin`.
