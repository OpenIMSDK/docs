<center>

    ## Get Conversation List

</center>

### Brief Description
- Obtain a list of conversations based on whether they are pinned and the order of message sending time.
### Request Method
- `post`
### Request URL
- `http://x.x.x.x:10002/msg/get_conversations_list`


### Header
|Header Name|Example Value|Optional|Type|Description|
|:----    |:-------    |:--- |---|------      |
|token|{{token}}|Yes|string|Administrator token|
|operationID|1646445464564|Yes|string|operationID for global link tracking|


### Request Example


```json
{
  "userID": "9906953281",
  "conversationIDs": "si_2699373280_9906953281"
}
```
|Field Name|Optional|Type|Description|
|:----    |:-------    |:--- |---|
|userID|Yes|string|Current user ID|
|conversationIDs|No|string|List of conversation IDs|

### Parameters Explanation for Successful Response


| Parameter Name    | Type   | Description                                                                                                                                                                                                   |
|:------------------|:-------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| errCode           | int    | Error code, 0 indicates success                                                                                                                                                                               |
| errMsg            | string | Brief error message, empty if there are no errors                                                                                                                                                             |
| errDlt            | errDlt | Detailed error message, empty if there are no errors                                                                                                                                                          |
| data              | object | Generic data object, see below for structure                                                                                                                                                                  |
| conversationElems | array  | N/A                                                                                                                                                                                                           |
| conversationID    | string | Conversation ID                                                                                                                                                                                               |
| unreadCount       | string | Unread message count                                                                                                                                                                                          |
| IsPinned          | string | Whether the conversation is pinned                                                                                                                                                                            |
| recvMsgOpt        | string | Group chat message disturbance-free option                                                                                                                                                                    |
| msgInfo           | object | Message content                                                                                                                                                                                               |
| serverMsgID       | string | Server message ID                                                                                                                                                                                             |
| clientMsgID       | string | Client message ID                                                                                                                                                                                             |
| sessionType       | string | Conversation type, indicating whether the sent message is a private chat or group chat; private chat is 1, normal group chat is 2, large group (read dissemination interface) is 3, notification session is 4 |
| recvID            | string | Receiver's ID                                                                                                                                                                                                 |
| SendID            | string | Sender's ID                                                                                                                                                                                                   |
| faceURL           | string | Avatar URL. In a private chat, when the sender is the current user, this field is the friend's avatar URL                                                                                                     |
| senderName        | string | Sender's nickname. In a private chat, when the sender is the current user, this field is the friend's nickname. In a group chat, this field is the corresponding sender's nickname                            |
| LatestMsgRecvTime | string | The receiving time of the last message                                                                                                                                                                        |
| msgFrom           | string | Message source, 100 from user sent, 200 from administrator sent or system broadcast notification, etc.                                                                                                        |
| contentType       | string | Message type, 101 for text, 102 for image.. refer to the message type format description document for details                                                                                                 |
| content           | string | Content                                                                                                                                                                                                       |
| groupID           | string | Group chat ID                                                                                                                                                                                                 |
| groupName         | string | Group chat name                                                                                                                                                                                               |
| groupFaceURL      | string | Group chat avatar URL                                                                                                                                                                                         |
| groupMemberCount  | string | Number of people in the group chat                                                                                                                                                                            |
