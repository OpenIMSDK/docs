<center>

## add_friend_response

</center>

### Brief Description
- Approve or reject the friend request initiated by fromUserID to toUserID.

### Request Method
- `post`

### Request URL
- `http://x.x.x.x:10002/friend/add_friend_response`

### Header
| Header Name  | Example Value | Optional | Type   | Description                                                |
|:------------ |:------------- |:-------- |:------ |:---------------------------------------------------------- |
| operationID  | 1646445464564 | Required | string | operationID is used for global link tracking               |
| token        | eyJhbxxxx3Xs  | Required | string | Administrator token                                        |

### Request Sample Parameters

```json
{
  "fromUserID": "11111111",
  "toUserID": "11111112",
  "handleResult": 1,
  "handleMsg": "agree"
}
```
| Field Name    | Optional | Type    | Description                             |
|:------------- |:-------- |:------- |:--------------------------------------- |
| fromUserID    | Required | string  | Request initiator                       |
| toUserID      | Required | string  | Request receiver                        |
| handleResult  | Required | int     | Handling result, 1: Agree; -1: Reject   |
| handleMsg     | Required | string  | Handling message                        |

### Successful Return Sample

```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### Successful Return Sample Explanation

| Parameter Name | Type   | Description                                                                                     |
|:-------------- |:------ |:----------------------------------------------------------------------------------------------- |
| errCode        | int    | Error code, 0 indicates success                                                                 |
| errMsg         | string | Brief error message, empty when no error                                                        |
| errDlt         | errDlt | Detailed error message, empty when no error                                                     |
| data           | object | General data object, the specific structure is shown below (Note: No data object is provided)   |

### Failed Return Sample
