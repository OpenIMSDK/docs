## General Field Descriptions

| Field Name     | Type   | String Length Limit | Description                                                     | Value Explanation                                                                                     |
| -------------- | ------ | ------------------- | ---------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| secret         | string | 32                  | OpenIM secret, from the `secret` field in `config/share.yml`     | String                                                                                                |
| platformID     | int    |                     | Device type when the user logs in                                | 1: iOS, 2: Android, 3: Windows, 4: OSX, 5: WEB, 6: Mini Program, 7: Linux, 8: AndroidPad, 9: iPad, 10: Admin|
| userID         | string | 64                  | Unique user ID within IM                                         | String                                                                                                |
| nickname       | string | 255                 | User nickname                                                    | String                                                                                                |
| faceURL        | string | 255                 | Profile picture URL                                              | URL link                                                                                              |
| ex             | string | 1024                | Extension field, recommend JSON-formatted string                 | String/JSON                                                                                           |
| operationID    | string |                     | Operation ID for issue tracking, unique per operation            | String                                                                                                |
| operatorUserID | string | 64                  | Operator’s ID, varies depending on the context                   | String                                                                                                |
| groupID        | string | 64                  | Unique group ID within IM                                        | String                                                                                                |
| sessionType    | int    |                     | Conversation type                                                | 1: Private, 3: Group, 4: System Notification                                                          |

## ConversationInfo

> Conversation information

| Field Name            | Type    | Max String Length Limit | Description                                                | Range    |
| --------------------- | ------- | ----------------------- | ---------------------------------------------------------- | -------- |
| ownerUserID           | string  |                         | Owner user ID of the conversation                           |
| conversationID        | string  |                         | Conversation ID                                            |
| recvMsgOpt            | int     |                         | Receive message options, 0: Receive; 1: Do not receive; 2: Receive without alerts |
| conversationType      | int     |                         | Conversation type, 1: Private chat, 3: Group chat          |
| userID                | string  |                         | Conversation user ID, applies when type is 1               |
| groupID               | string  |                         | Group ID, applies when type is 3                           |
| isPinned              | boolean |                         | Whether the conversation is pinned                         |
| attachedInfo          | string  |                         | OpenIM’s extension field                                   |
| isPrivateChat         | boolean |                         | Whether private chat is enabled                            |
| groupAtType           | int     |                         | Group announcement notification type                       |
| ex                    | string  |                         | Extension field for user usage                             |
| burnDuration          | int     |                         | Message burn duration                                      |
| minSeq                | int     |                         | Minimum sequence number retrievable in this conversation   |
| maxSeq                | int     |                         | Maximum sequence number retrievable in this conversation   |
| msgDestructTime       | int     |                         | Message self-destruct interval                             |
| latestMsgDestructTime | int     |                         | Last message destruct time                                 |
| isMsgDestruct         | boolean |                         | Whether timed message destruction is enabled               |
