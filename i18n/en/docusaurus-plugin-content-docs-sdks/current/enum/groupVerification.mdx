# GroupVerification

:::info

Group entry verification options.

:::

## Group Entry Verification Options {#GroupVerification}

| Enum Value | Description                                                                                         |
| ---------- | --------------------------------------------------------------------------------------------------- |
| 0          | Application to join requires group owner or admin approval; group members can invite directly.      |
| 1          | Application to join requires group owner or admin approval; only group owner and admins can invite directly, while regular member invites require group owner or admin approval. |
| 2          | Join directly without verification.                                                                 |
