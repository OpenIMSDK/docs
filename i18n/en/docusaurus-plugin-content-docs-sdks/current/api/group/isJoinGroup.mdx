<TabItem value="uni-app">

### Function Prototype

```ts showLineNumbers
IMSDK.asyncApi('isJoinGroup', operationID: string, groupID: string): Promise<boolean>
```

### Input Parameters

| Parameter Name | Parameter Type | Required | Description  |
| -------------- | -------------- | -------- | ------------ |
| operationID    | string         | Yes      | Operation ID for issue tracking, unique; recommended to use current time and a random number |
| groupID        | string         | Yes      | Group ID    |

### Return Results

> The function is transformed into a Promise using the `openim-uniapp-polyfill` package. When calling, use `then` and `catch` to determine and handle successful and failed callbacks.

| Parameter Name | Parameter Type                                  | Description       |
| -------------- | ------------------------------------------------ | ----------------- |
| Promise.then() | Promise<boolean\>                                    | Success callback  |
| Promise.catch() | Promise<[CatchResponse](/class/response.mdx)\>   | Failure callback  |

### Code Example

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

IMSDK.asyncApi('isJoinGroup', IMSDK.uuid(), 'groupID')
  .then(() => {
    // Successful call
  })
  .catch(({ errCode, errMsg }) => {
    // Failed call
  });
```
