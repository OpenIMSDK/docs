# onFriendApplicationRejected

## Feature Introduction

:::info

When a friend application is rejected, both the initiator and the receiver of the application will receive this callback.

:::

### Call Example

```js showLineNumbers

// or
// import { OpenIMSDK, CbEvents } from 'open-im-sdk';

  // data - Friend Application
```
