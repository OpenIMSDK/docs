###  SearchFriendsParam

| Field Name        | Data Type  | Description                        |
| --------------    | --------   | ----------------------- |
| KeywordList       | string[]   |  Search Keyword, Currently Only Support Single Keyword Search, Cannot Be Empty  |
| IsSearchUserID    | bool       |  Whether To Search UserID By Keyword |
| IsSearchNickname  | bool       |  Whether To Search By Keyword For Nickname, Default Is False. |
| IsSearchRemark    | bool       |  Whether To Search By Keyword For Remarks, Default Is False.  |
